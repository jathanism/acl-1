class PMTree
    PMTree (prefix match tree) is a binary tree with the following features:
1) Each terminal node has a label, being one of the decisions (e.g. 'permit' or 'deny')
2) Each non-terminal node has a label 'n/a'
3) Each non-terminal node n has two children node l and r. the left edge (n,l) has label 
'0' and the right edge (n,r) has label '1'
A PMTree can be constructed via a prefix priority rule set f. For example:
f = r1: {101***} -> permit
    r2: {0*****} -> permit
    r3: {******} -> deny
